framecraft

framecraft is an LLM skill and plugin for creating polished demo videos. You describe what you want; your LLM writes the HTML scenes, narration, and config, then framecraft renders everything into a real 1920x1080 MP4.

The user is trying to debug CI output where the rendered video is correct locally but the second scene loses voiceover in the generated artifact.
